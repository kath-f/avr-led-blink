# avr-led-blink
Making an LED flash is like the hello world of messing with microcontrollers.

This should work with any AVR MCU i'm pretty sure, but i only have an arduino, and no breadboard, and, thus the schematics uses an Arduino Uno R3 to reflect my setup, because i don't know how to hook up an atmega328p standalone, and right now i don't have the resources to try that.

### DISCLAIMER
If you're not using an arduino like me you might want to change some settings in the Makefile to reflect your setup
